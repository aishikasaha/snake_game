# snake_game
This projects uses basic pygame library to create the old snake_ladder game

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dependencies

```bash
import math
import random
import pygame
import tkinter 
from tkinter import messagebox
```

## Usage

```bash
python3 game.py
```
